# Currant Design System

Style guide for the Currant financial dashboard app.

Source: https://v0-currant-financial-dashboard.vercel.app/

---

## Brand Colors

| Name             | Hex       | Usage                        |
|------------------|-----------|------------------------------|
| Currant Red      | `#8B2942` | Primary brand color          |
| Currant Burgundy | `#5C1A2A` | Deep accent                  |
| Currant Black    | `#3D2438` | Black currant inspired       |
| Currant Leaf     | `#3D8B4F` | Fresh green accent           |
| Currant Cream    | `F7F3E8`  | Soft neutral background      |

## UI Color Tokens

| Token            | Usage                        |
|------------------|------------------------------|
| `--primary`      | Main actions                 |
| `--secondary`    | Secondary elements           |
| `--accent`       | Highlights & success         |
| `--destructive`  | Errors & warnings            |
| `--background`   | Page background              |
| `--foreground`   | Primary text                 |
| `--muted`        | Subtle backgrounds           |
| `--border`       | Dividers & outlines          |
| `--chart-1` – `--chart-5` | Data visualization |

---

## Typography

| Role         | Font           | Usage                                  |
|--------------|----------------|----------------------------------------|
| Display      | Fraunces       | Headlines, hero text, brand moments    |
| Sans-serif   | DM Sans        | Body text, UI elements, navigation     |
| Monospace    | JetBrains Mono | Numbers, financial data, code          |

### Type Scale

| Token      | Size  | Usage                  |
|------------|-------|------------------------|
| Display XL | 7xl   | Hero / display text    |
| Heading 1  | 5xl   | Page titles            |
| Heading 2  | 3xl   | Section headings       |
| Heading 3  | xl    | Sub-section headings   |
| Large Body | lg    | Emphasized body text   |
| Body       | base  | Regular body text      |
| Small      | sm    | Labels, captions       |

## Components

### Buttons

Variants: `primary`, `secondary`, `outline`, `ghost`, `link`, `destructive`

Sizes: `large`, `default`, `small`

### Inputs

States: `default`, `with value`, `disabled`

### Badges

Variants: `default`, `secondary`, `outline`, `destructive`, `income`, `expense`

### Cards

Patterns: Net Worth, Monthly Income, Monthly Expenses
